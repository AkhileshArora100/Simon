How To Play

Press any key/ click anywhere to start the Game. The Skill Level increases by one each time.

Simon will give the first signal by pressing a button. Repeat the signal by pressing the same color button or the key that is written on it.

Simon will duplicate the first signal and add one. Repeat these two signals by pressing the same color button in the same order.

Simon will duplicate these first two signals and add one. Similarly the game will keep adding a signal each time you clear a level.

Continue playing as long as you can repeat each sequence of signals correctly. After the 5th, 9th and 13th signals in a sequence, Simon automatically speeds up.

If you fail to repeat a sequence exactly, Simon responds with a "RAZZ" sound. This means you've lost, and the sequence of signals ends.
